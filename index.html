Questions fréquentes

Comment accéder à l'application après achat ?+
Après votre achat sur Gumroad, vous recevez immédiatement par email un code d'accès unique. Revenez sur cette page, cliquez sur "J'ai déjà un code" et entrez votre code pour accéder à l'application.
C'est quoi un PMS exactement ?+
Le Plan de Maîtrise Sanitaire est un document obligatoire pour tous les établissements alimentaires en France. Il décrit les mesures prises pour assurer la sécurité alimentaire, conformément au règlement CE 852/2004.
Qui est concerné par le PMS ?+
Tous les professionnels de l'alimentation : restaurants, hôtels, traiteurs, cantines, boulangeries, épiceries, food trucks, etc.
Le PDF est-il accepté par la DDPP ?+
Oui, le document généré respecte les exigences du règlement CE 852/2004. Il couvre toutes les sections requises lors d'un contrôle sanitaire.
Puis-je mettre à jour mon PMS ?+
Oui, vous pouvez créer autant de versions que nécessaire grâce à la sauvegarde automatique. Recommandé : une mise à jour par an minimum.
